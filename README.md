### Dashboard Purpose

This dashboard helps HR teams move from simple reporting to predictive decision-making.

Instead of only knowing how many employees left, HR can now identify:

* Which employees may leave
* Which departments have higher attrition
* Which job roles are risky
* How overtime affects attrition
* What HR action should be taken for each risk category

### Key Dashboard Insight

Employees working overtime, employees in Sales roles, younger employees, and low-income employees show higher attrition risk.

HR should focus on proactive retention strategies for these employee groups.
